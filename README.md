# 🚖 FairRide – Smart Ride Booking Platform

FairRide is a modern **Progressive Web Application (PWA)** that connects riders and drivers through a simple, responsive, and user-friendly interface. The application is built using **HTML, CSS, and JavaScript**, offering separate Rider and Driver dashboards within a single application while supporting offline capabilities and mobile-friendly performance.

---

## 📖 Project Overview

FairRide aims to simplify urban transportation by providing an intuitive platform for booking and managing rides. The application allows users to switch between Rider and Driver modes, making it suitable for demonstrating ride-sharing workflows in a lightweight web application.

Designed as a static web application, FairRide can be deployed easily on platforms like **GitHub Pages** and **Vercel** without requiring a backend server.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/yoganjanaLeela-debug/Fair-Raid.git
```


## 🔄 Application Workflow

1. Launch the FairRide application.
2. Choose either **Rider** or **Driver** mode.
3. Riders can simulate booking rides.
4. Drivers can manage and accept ride requests.
5. Interactive maps provide location visualization.
6. QR code functionality supports quick ride interactions.
7. The application works offline after the initial load through PWA support.
